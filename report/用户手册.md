# 六、用户手册

## 1. 安装部署说明

### 1.1 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows / macOS / Linux |
| Python版本 | Python 3.8 及以上 |
| 数据库 | SQLite3（内置，无需额外安装） |
| 浏览器 | Chrome / Edge / Firefox（推荐最新版） |

### 1.2 部署步骤

**第一步：准备项目文件**

将以下文件放置于同一目录下：

- `main.py`（主程序文件）
- 程序运行后会自动生成 `campus_team_v11_clean.db`（数据库文件）

**第二步：安装依赖库**

打开终端（命令行），进入项目目录，执行以下命令：

```bash
pip install fastapi uvicorn
```

> 注：sqlite3 和 hashlib 为 Python 内置模块，无需额外安装。

**第三步：启动服务**

在终端中执行：

```bash
python main.py
```

启动成功后，控制台会显示：

=================================================== 🚀 校园组队系统启动成功！ 🌐 访问地址: http://127.0.0.1:8002


**第四步：访问系统**

打开浏览器，访问地址：`http://127.0.0.1:8002`

### 1.3 常见问题

| 问题 | 解决方法 |
|------|----------|
| 端口被占用 | 修改代码末尾的 `port=8002` 为其他端口（如 8080） |
| 无法连接数据库 | 检查当前目录是否有写入权限 |
| 页面样式异常 | 检查网络是否能访问 CDN（tailwindcss、marked.js） |

## 2. 操作指南

### 2.1 注册与登录

**注册账号**

1. 访问首页，点击"没有账号？去注册"
2. 填写：学号（12位数字）、姓名/昵称、手机号、密码
3. 点击"注册并登录"完成注册

**登录系统**

- 方式一：使用手机号 + 密码登录
- 方式二：使用学号（12位）+ 密码登录

**修改密码**

1. 点击大厅右上角"修改密码"按钮
2. 输入学号、手机号进行身份验证
3. 输入原密码和新密码，确认后提交

### 2.2 完善个人信息

1. 点击大厅右上角的"个人信息"按钮（或头像）
2. 填写以下信息（带 * 为必填）：
   - 学院、专业、班级
   - 技能/兴趣标签（如：Python，Java，前端），用于智能推荐
   - QQ号、微信号（方便组队联系）
   - 个人荣誉（如：国家奖学金）
   - 个性签名
3. 点击"保存并关闭"

> 💡 提示：技能标签填写越详细，智能推荐越精准！

### 2.3 发布招募项目

1. 点击大厅顶部的"➕ 发布招募"按钮
2. 填写项目信息：
   - 项目名称
   - 已有人数 / 共需人数
   - 技术标签（逗号分隔，如：Python，数据库）
   - 项目描述（支持 Markdown 语法）
3. 点击"立即发布"，项目将显示在公共招募大厅

### 2.4 管理我的招募

点击"📝 我的招募"按钮

对每个项目可执行以下操作：

| 操作 | 说明 |
|------|------|
| 隐藏招募 | 仅自己可见，其他人看不到此项目 |
| 停止招募 | 项目状态变为"已满员/截止"，不可再申请 |
| 删除招募 | 永久删除项目（不可恢复） |

### 2.5 申请加入项目

1. 在公共招募大厅浏览项目
2. 点击项目卡片中的"申请加入"按钮
3. 队长审批前可点击"我的申请"中的"撤销申请"取消

### 2.6 队长审批申请

1. 点击"🛡️ 队长审批"按钮
2. 查看待审核成员列表，可看到申请者的荣誉信息
3. 选择"同意进组"或"婉拒"
4. 审批后，申请记录会保留在历史中
5. 当队伍满员时，项目状态自动变为"已截止"

### 2.7 消息聊天

**私聊功能**

1. 在大厅点击任意用户的头像
2. 点击"💬 发起聊天"
3. 输入消息后发送

**项目群聊**

- 入队成功后，可通过"我的申请"或项目卡片进入群聊
- 群聊内所有队员和队长均可收发消息
- 队长可在右侧成员面板中"移出"队员

**消息管理**

- 未读消息会在"消息"按钮和对应会话上显示红点
- 点击会话列表右侧的"🗑️"按钮可清空聊天记录（仅自己不可见）

### 2.8 搜索与推荐

**关键词搜索**

在搜索框中输入项目名称或描述关键词，点击"快速搜索"

**标签筛选**

在标签筛选框中输入标签（如：Python），按标签过滤项目

**智能推荐**

- 系统根据您填写的"技能/兴趣标签"自动匹配推荐项目
- 推荐结果显示在右侧"猜你喜欢"面板
- 匹配度越高，排名越靠前

### 2.9 退出与多端互斥

- 点击"退出"按钮安全登出
- 同一账号在另一设备登录时，原设备会被强制下线并弹出提示
